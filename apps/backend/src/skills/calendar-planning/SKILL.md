# Calendar Planning

Use this skill to help managers organize their calendar so that meetings, deep work, and administrative tasks are balanced effectively.

## When to use

- A manager needs help fitting new commitments into a packed schedule.
- You want to propose an optimal weekly calendar structure.
- Someone asks how to protect focus time while handling frequent meetings.

## Input

Provide existing calendar commitments, recurring meetings, deadlines, preferred working hours, and time-zone constraints if applicable.

## Output format

1. **Weekly overview** - High-level distribution of meeting time vs. focus time vs. admin time.
2. **Day-by-day plan** - Table with day, time blocks, and activity type.
3. **Protected blocks** - Explicit deep-work slots marked as non-negotiable.
4. **Scheduling tips** - Two to three recommendations for keeping the plan sustainable.

## Guidelines

- Cluster meetings together to reduce context-switching.
- Reserve the first or last 30 minutes of the day for email and admin.
- Flag any conflicts or over-commitments immediately.
- Respect the manager's stated preferences for meeting-free mornings or afternoons.
- If the calendar is overloaded, suggest meetings to decline or delegate.
