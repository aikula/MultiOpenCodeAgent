# Executive Summary

Use this skill when you need to distill a lengthy document, report, or thread into a brief that a busy executive can absorb in under two minutes.

## When to use

- A manager shares a long report and asks "what does this say?"
- You need to summarize a multi-page proposal before a decision meeting.
- A stakeholder requests a high-level overview without the details.

## Input

Provide the full text or a clear reference to the source material.

## Output format

1. **Bottom line** - One sentence stating the most important conclusion.
2. **Key findings** - Three to five bullet points capturing essential insights.
3. **Risks or open questions** - Up to three items that need attention.
4. **Recommended next steps** - Two to three actionable items.

## Guidelines

- Keep the total output under 200 words.
- Use plain language; avoid jargon unless the audience expects it.
- Prioritize information that drives decisions.
- If the source is ambiguous, flag the ambiguity rather than guessing.
