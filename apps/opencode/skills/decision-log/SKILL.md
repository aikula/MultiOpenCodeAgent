# Decision Log

Use this skill when the user asks to log, document or structure a decision.

Return:

1. Decision title
2. Context and problem statement
3. Alternatives considered
4. Rationale for chosen option
5. Expected outcome
6. Risks and mitigations
7. Follow-up actions

Be precise. Distinguish between firm decisions and tentative preferences.
