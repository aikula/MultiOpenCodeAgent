# Email Draft

Use this skill when the user asks to write, draft or compose an email.

Return:

1. Subject line (concise, specific)
2. Recipient suggestion if not specified
3. Email body (professional, clear, action-oriented)
4. Call to action / next steps
5. Tone note (formal, semi-formal, follow-up)

Default language: Russian unless user writes in English. Keep emails under 200 words.
