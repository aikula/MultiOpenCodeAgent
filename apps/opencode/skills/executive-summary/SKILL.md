# Executive Summary

Use this skill when the user asks to summarize, condense or get a brief of content.

Return:

1. One-paragraph overview
2. Key points (3-5 bullets)
3. Decisions or outcomes
4. Open questions or risks
5. Recommended next steps

Be concise. Maximum one page equivalent. Prioritize action items over background.
