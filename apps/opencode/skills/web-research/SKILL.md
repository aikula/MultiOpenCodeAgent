# Web Research

Use this skill when the user asks to research, investigate or find information.

Approach:

1. Clarify the research question if ambiguous
2. Use context7 MCP tools for current documentation
3. Use gh_grep MCP tools for code examples
4. Synthesize findings into structured answer

Return:

1. Research question (confirmed)
2. Key findings (bulleted)
3. Sources used
4. Confidence level
5. Gaps remaining

Always cite sources. Distinguish between facts and opinions.
