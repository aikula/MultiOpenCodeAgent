# Meeting Followup

Use this skill when the user asks for follow-up after a meeting or discussion.

Return:

1. Summary of what was discussed (brief)
2. Action items with owners and deadlines
3. Suggested reminders for key deadlines
4. Follow-up email draft if applicable
5. Decisions that need confirmation

Create reminders and calendar events directly when possible. Draft emails inline.
