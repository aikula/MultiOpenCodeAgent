# GitHub Code Search

Use this skill when the user asks for code examples, implementation patterns or reference code.

Use gh_grep MCP tools to search public GitHub.

Return:

1. Search query used
2. Relevant code snippets (2-5 examples)
3. Repository and file source for each
4. Brief explanation of how each example works
5. Recommended approach based on findings

Focus on well-maintained repositories. Note the license if relevant.
