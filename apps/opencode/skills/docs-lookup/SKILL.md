# Documentation Lookup

Use this skill when the user asks about a technology, library, API or framework.

Use context7 MCP tools to find current documentation.

Return:

1. Technology/library name and version
2. Relevant documentation excerpt
3. Code example if applicable
4. Common gotchas or migration notes
5. Link to official docs

Prioritize current docs over cached knowledge. If context7 is unavailable, say so.
