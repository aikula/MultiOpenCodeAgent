# Find Context

Use this skill when the user asks to find prior discussions, decisions, or context.

Return:

1. Summary of relevant prior items
2. Key decisions and commitments found
3. Unresolved items
4. Source references (date, session)

Quote source message dates. If no matches, say so explicitly.
