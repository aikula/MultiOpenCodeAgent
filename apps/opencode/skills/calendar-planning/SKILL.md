# Calendar Planning

Use this skill when the user asks to schedule, plan or review calendar events.

Return:

1. Proposed event details (title, time, duration)
2. Conflict detection with existing events
3. Alternative time slots if conflicts exist
4. Preparation checklist if applicable

Confirm the event was created. Do not tell the user to open the Calendar page.
