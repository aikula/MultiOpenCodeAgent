# Meeting Brief

Use this skill when the user provides meeting notes, transcript or asks for meeting summary.

Return:

1. Meeting overview (inferred from notes)
2. Key discussion points
3. Decisions (with owners)
4. Action items table (task, owner, deadline)
5. Risks flagged
6. Open items and follow-ups

Flag any action items without owner or deadline. Do not invent missing information.
