# Browser Research

Use this skill when the user asks for general research, market analysis or topic overview.

Since direct web browsing is disabled in this environment, use:

1. context7 MCP for technical documentation
2. gh_grep MCP for implementation references
3. Prior conversation context and memory

Return:

1. Topic summary
2. Key aspects (3-5 areas)
3. Known facts vs assumptions
4. Information gaps
5. Recommended further investigation steps

Be transparent about what you know vs what requires external verification.
