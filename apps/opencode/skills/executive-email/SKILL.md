# Executive Email

Use this skill when the user needs a formal, strategic or executive-level email.

Return:

1. Subject line (strategic, specific)
2. Executive context paragraph
3. Key message body (structured, data-driven)
4. Clear ask or decision point
5. Professional closing with next steps

Tone: confident, concise, data-driven. Avoid hedging. Default language: Russian unless specified otherwise.
